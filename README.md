# Zero1Matrix

Local-first / Offline-first 易經起卦與解卦系統施工包。

本包目標是讓 OSS 120B 或工程團隊可以直接依規格施工，建立一個：

- 原生 HTML / CSS / JavaScript
- 無框架、無後端、無會員、無追蹤
- 支援 file:// 完整核心功能
- http/https 下支援完整 PWA
- 繁體中文
- 深色 RWD 高質感
- 原文 + 白話解釋
- 12 類問事分類
- 三層解卦：一句話、簡解、深解
- 多維度卦象分數
- 本機歷史紀錄、JSON 匯入匯出、HTML 報告、列印 PDF
- 獨立 tools 驗證頁
- 完整詞庫 schema、驗證規則、測試案例與銅錢素材

## 重要原則

使用者任何資料都不回傳 server。

server 若存在，只負責提供靜態檔案。  
不需要 API、不需要資料庫、不需要登入、不呼叫 AI、不放追蹤 SDK。

## file:// 模式

file:// 模式使用 `.data.js` 載入詞庫，不使用 fetch JSON。  
核心功能必須可用：

- 起卦
- 解卦
- 讀取詞庫
- 歷史紀錄基本功能
- tools 驗證頁
- 匯出 HTML 報告

## 資料維護策略

原始資料可用 JSON 維護。  
正式發佈時產出 `.data.js`，供 file:// 與純前端載入。

## 目前包內容

這是一份完整施工包，包含：

- 施工書
- 專案結構
- 詞庫 schema
- 初版可執行骨架
- 全 64 卦基礎資料
- 384 爻資料骨架與前 8 卦示範資料
- 4096 本卦變卦資料生成規格與種子資料
- 12 類問事分類規格
- 多維度分數規格
- 驗證工具規格
- 起卦演算法規格
- 測試 fixtures
- 銅錢 SVG 素材

## 注意

百萬字級完整白話詞庫需要依本包 schema 逐批擴寫與校驗。  
本包提供可施工、可驗證、可擴充的地基與初版資料，並明確標示哪些資料需要補齊。
