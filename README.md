# 易卦

> 不問天命，只觀時勢。

易卦是一套以《易經》為基礎的本機優先起卦與解卦工具。專案使用原生 HTML、CSS 與 JavaScript 建置，不需要後端、帳號或資料庫；起卦紀錄與設定保存在使用者自己的瀏覽器中，可直接離線使用，也可部署成靜態網站與 PWA。

這個專案希望把卦象從單一吉凶判斷，轉化為可閱讀、可追溯的決策參考：保留卦辭與爻辭原文，同時提供白話解釋、問事分類、動爻分析、風險提醒與具體行動方向。

## 核心功能

- 三枚銅錢起卦，支援逐爻動畫與快速模式
- 每日一卦，以當日日期產生固定結果
- 手動排卦，可直接輸入六爻的 6／7／8／9
- 12 種問事分類：一般、工作事業、感情關係、財務金錢、人際合作、家庭親人、學習考試、身心狀態、重大決策、創業經營、官非合約、心境修行
- 顯示本卦、變卦、動爻、互卦、錯卦與綜卦
- 卦辭、象辭、爻辭原文與繁體中文白話解釋
- 一句話提示、簡解與深解等不同閱讀層次
- 六維度評分：局勢明朗度、行動適合度、風險壓力度、變化強度、人和支援度、時機成熟度
- 依動爻與問事分類提供專屬解讀、行動建議、風險提醒與反思問題
- 歷史紀錄搜尋、分頁、刪除及 JSON 匯入匯出
- 單筆解卦可匯出 JSON 或 HTML 報告
- 報告檢視與列印／另存 PDF
- 深色響應式介面，可在桌面與行動裝置使用

## 隱私與離線設計

易卦採用 local-first／offline-first 架構：

- 不需要登入或建立帳號
- 不將起卦內容與歷史紀錄上傳到伺服器
- 不使用追蹤 SDK
- `file://` 模式可直接載入 `.data.js` 詞庫並使用核心功能
- 透過 HTTP／HTTPS 部署時可註冊 Service Worker，提供 PWA 體驗
- 若使用靜態伺服器，伺服器只負責傳送專案檔案

所有歷史紀錄均儲存在瀏覽器的 `localStorage`。清除瀏覽器網站資料前，建議先從歷史紀錄頁匯出 JSON 備份。

## 快速開始

### 直接離線開啟

下載或複製專案後，直接以瀏覽器開啟 `index.html`。核心起卦、解卦、歷史紀錄與報告功能不依賴網路。

### 使用靜態伺服器

若要測試 Service Worker、PWA 或較完整的瀏覽器部署行為，可在專案根目錄啟動任意靜態伺服器。例如：

```bash
python -m http.server 8080
```

接著開啟：

```text
http://localhost:8080/
```

HTTPS 環境下會自動註冊 `sw.js`。

## 使用方式

1. 在首頁選擇「三枚銅錢」、「每日一卦」或「手動排卦」。
2. 選擇最接近問題性質的問事分類。
3. 完成六爻後查看本卦、變卦、動爻與六維度分數。
4. 依序閱讀一句話提示、簡解、深解、行動建議與風險提醒。
5. 將結果儲存到本機，或匯出 JSON／HTML 報告。

易卦提供的是情勢整理與反思框架，不取代醫療診斷、法律意見或財務專業判斷。

## 技術特點

- 原生 HTML／CSS／JavaScript，無前端框架
- 詞庫透過 `window.Zero1MatrixData` 載入，兼容 `file://`
- 六爻資料採 bottom-up 儲存，`value[0]` 為初爻
- 6／8 為陰爻，7／9 為陽爻；6／9 為動爻
- 六十四卦以六爻陰陽組合進行唯一映射
- 評分保留來源與調整軌跡，便於檢查各項分數如何形成
- 靜態資源即可部署，不需要 API 或伺服器端運算
